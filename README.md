🎵 Sistema de Música

📖 Sobre o Projeto

O Sistema de Música é uma aplicação desenvolvida em Java utilizando os conceitos de Programação Orientada a Objetos (POO). O objetivo do sistema é permitir o gerenciamento de usuários, artistas, músicas, álbuns e playlists, simulando funcionalidades presentes em plataformas de streaming musical.

O projeto está sendo desenvolvido utilizando a arquitetura MVC (Model-View-Controller), promovendo uma melhor organização do código e separação de responsabilidades.

O README será descrito completamente, após a finalização do projeto.
